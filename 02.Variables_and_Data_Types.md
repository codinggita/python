# Variables and Data Types in Python

---

# 1. Introduction

In programming, data must be stored so it can be reused later.
In Python, we store data inside **variables**.

Variables are one of the most important foundations of programming.
Without variables, you cannot perform calculations, take input, or build logic.

---

# 2. What is a Variable?

A variable is a **name given to a value**.

It allows us to store information and use it later.

Example:

```python
name = "Aarav"
age = 21
```

Here:

* `name` stores text
* `age` stores a number

To display them:

```python
print(name)
print(age)
```

---

# 3. Rules for Naming Variables

You must follow these rules:

1. A variable name can contain letters, numbers, and underscore (`_`)
2. It cannot start with a number
3. It cannot use Python keywords (like `if`, `for`, `while`)
4. Python is case-sensitive (`age` and `Age` are different)

Valid examples:

```python
student_name = "Riya"
marks1 = 90
total_score = 450
```

Invalid examples:

```python
1marks = 90        # Cannot start with number
class = "BCA"      # 'class' is a keyword
```

---

# 4. Understanding Dynamic Typing

Python automatically understands the type of data you assign.

Example:

```python
x = 10      # integer
x = "Hello" # now string
```

You do not need to declare the data type separately.
Python decides it automatically.

This is called **dynamic typing**.

---

# 5. Core Data Types for Beginners

We focus only on the most important types.

---

## 5.1 Integer (`int`)

Whole numbers without decimals.

```python
age = 18
total_marks = 450
```

---

## 5.2 Float (`float`)

Numbers with decimal values.

```python
price = 99.99
temperature = 36.5
```

---

## 5.3 String (`str`)

Text values written inside quotes.

```python
name = "Kiran"
city = 'Mumbai'
```

Both single and double quotes are allowed.

---

## 5.4 Boolean (`bool`)

Represents only two values:

* True
* False

Example:

```python
is_student = True
is_logged_in = False
```

Booleans are mainly used in conditions (which you will learn later).

---

# 6. Checking Data Type Using `type()`

You can check the type of a variable using `type()`.

Example:

```python
age = 22
print(type(age))
```

Output:

```
<class 'int'>
```

Another example:

```python
name = "Python"
print(type(name))
```

---

# 7. Reassigning Variables

You can change the value of a variable anytime.

```python
score = 50
score = 75
print(score)
```

The latest value will overwrite the previous one.

---

# 8. Multiple Assignment

Python allows assigning multiple variables in one line.

Example:

```python
a, b, c = 10, 20, 30
```

Or assigning same value:

```python
x = y = z = 5
```

---

# 9. Common Beginner Mistakes

1. Forgetting quotes in strings:

```python
name = Rahul   # Error
```

2. Confusing number and string:

```python
age = "20"
print(age + 5)  # Error
```

3. Using keywords as variable names.
