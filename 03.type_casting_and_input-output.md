# Type Casting and Input/Output in Python

---

# 1. Introduction

In real programs, we must:

* Take data from users
* Display results
* Convert data from one type to another

These concepts make programs interactive and practical.

---

# 2. Output in Python – `print()`

The `print()` function is used to display output on the screen.

Basic example:

```python
print("Welcome to Python")
print(100)
```

You can also print variables:

```python
name = "Aarav"
age = 20
print(name)
print(age)
```

---

## Printing Multiple Values

```python
city = "Delhi"
year = 2026
print(city, year)
```

Python automatically adds a space between values.

---

## Using f-Strings (Formatted Output)

F-strings allow you to insert variables inside text.

```python
name = "Riya"
age = 19
print(f"My name is {name} and I am {age} years old.")
```

This is the cleanest way to format output.

---

# 3. Input in Python – `input()`

The `input()` function allows users to enter data.

Example:

```python
name = input("Enter your name: ")
print(name)
```

Whatever the user types becomes the value of the variable.

---

## Important Concept: Input Always Returns a String

Even if the user types a number, Python treats it as a string.

Example:

```python
age = input("Enter your age: ")
print(type(age))
```

Output:

```
<class 'str'>
```

This is why type casting is important.

---

# 4. Type Casting

Type casting means converting one data type into another.

Python provides built-in functions for this.

---

## Converting to Integer

```python
age = int(input("Enter your age: "))
print(type(age))
```

Now `age` becomes an integer.

---

## Converting to Float

```python
price = float(input("Enter price: "))
print(type(price))
```

---

## Converting to String

```python
num = 100
text = str(num)
print(type(text))
```

---

# 5. Why Type Casting is Necessary

Consider this example:

```python
num1 = input("Enter first number: ")
num2 = input("Enter second number: ")
print(num1 + num2)
```

If user enters:

```
10
20
```

Output will be:

```
1020
```

Because they are strings.

Correct way:

```python
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))
print(num1 + num2)
```

Now output will be:

```
30
```

---

# 6. Common Beginner Mistakes

1. Forgetting to convert input to integer
2. Trying to add string and number together
3. Writing `Int()` instead of `int()` (Python is case-sensitive)
