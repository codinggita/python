# Operators in Python

---

# 1. Introduction

Operators are used to perform operations on variables and values.

For example:

* Adding numbers
* Comparing values
* Checking multiple conditions

Operators are essential for building logic in programs.

In this article, we will cover only the most important beginner-level operators.

---

# 2. Arithmetic Operators

These operators perform mathematical calculations.

| Operator | Meaning             |
| -------- | ------------------- |
| `+`      | Addition            |
| `-`      | Subtraction         |
| `*`      | Multiplication      |
| `/`      | Division            |
| `%`      | Modulus (remainder) |

---

## Addition (`+`)

```python
a = 10
b = 5
print(a + b)
```

Output:

```
15
```

---

## Subtraction (`-`)

```python
print(10 - 3)
```

Output:

```
7
```

---

## Multiplication (`*`)

```python
print(4 * 5)
```

Output:

```
20
```

---

## Division (`/`)

```python
print(10 / 2)
```

Output:

```
5.0
```

Note: Division always returns a float.

---

## Modulus (`%`)

Returns remainder after division.

```python
print(10 % 3)
```

Output:

```
1
```

Useful for checking even/odd:

```python
num = 8
print(num % 2)
```

If result is 0 → number is even.

---

# 3. Comparison Operators

Used to compare two values.
Result is always `True` or `False`.

| Operator | Meaning               |
| -------- | --------------------- |
| `==`     | Equal to              |
| `!=`     | Not equal             |
| `>`      | Greater than          |
| `<`      | Less than             |
| `>=`     | Greater than or equal |
| `<=`     | Less than or equal    |

---

Example:

```python
print(10 > 5)
print(5 == 5)
print(7 != 3)
```

Output:

```
True
True
True
```

These operators are mainly used in conditions.

---

# 4. Logical Operators

Used to combine multiple conditions.

| Operator | Meaning                      |
| -------- | ---------------------------- |
| `and`    | Both conditions must be True |
| `or`     | At least one must be True    |
| `not`    | Reverses result              |

---

## AND

```python
age = 20
print(age > 18 and age < 25)
```

Returns `True` if both conditions are true.

---

## OR

```python
print(10 > 5 or 2 > 8)
```

Returns `True` if at least one is true.

---

## NOT

```python
print(not True)
```

Output:

```
False
```

---

# 5. Operator Precedence (Basic Idea)

Python follows mathematical rules:

1. Multiplication and division first
2. Then addition and subtraction

Example:

```python
print(10 + 5 * 2)
```

Output:

```
20
```

Because multiplication happens first.

You can use brackets to control order:

```python
print((10 + 5) * 2)
```

Output:

```
30
```

---

# 6. Common Beginner Mistakes

1. Using `=` instead of `==` in comparisons
2. Forgetting type conversion in arithmetic
3. Confusing `%` with division
4. Mixing string and integer

Example mistake:

```python
age = "20"
print(age + 5)  # Error
```
